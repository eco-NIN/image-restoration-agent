# 登录注册页改动记录（20260502_161323）

## 改动目标
为 React + FastAPI 图像复原系统新增现代化登录注册入口页（左右分栏、卡片式、简洁科技风），并接入后端账号体系（bcrypt + JWT）。

## 约束执行
- 未改动 4KAgent 目录。
- 保持原有工作台、历史页、任务处理主逻辑不变。

## 后端改动
1. 文件：backend/app/main.py
- 新增 users 表初始化（id, username, password, created_at）。
- 新增 /api/register。
- 新增 /api/login。
- 新增 bcrypt 密码加密与校验。
- 新增 JWT 签发逻辑（HS256，默认24小时）。

2. 文件：requirements.txt
- 新增依赖：bcrypt、PyJWT。

## 前端改动
1. 新增认证模块
- frontend/src/auth/auth.js：token/username 本地存取。
- frontend/src/components/Toast.jsx：toast 提示组件（替代 alert）。
- frontend/src/components/ProtectedRoute.jsx：路由守卫。

2. 新增登录注册页
- frontend/src/pages/AuthPage.jsx
- 左侧品牌区：
  - 标题：4KAgent 图像复原系统
  - 副标题：让低质量图像重获细节与清晰
- 右侧表单区：登录/注册切换，平滑过渡，卡片式样式统一。

3. 接口接入
- frontend/src/services/apiClient.js
  - 新增 registerUser
  - 新增 loginUser

4. 路由接入
- frontend/src/App.jsx
  - 新增 /auth 路由
  - 系统主路由包裹 ProtectedRoute（未登录自动跳转 /auth）

5. 顶部退出
- frontend/src/components/AppLayout.jsx
  - 新增“退出”按钮
  - 显示当前用户名

## 使用说明
1. 安装后端依赖（新增 bcrypt 与 PyJWT）：
   pip install -r requirements.txt
2. 启动后端（保持 8001）
3. 启动前端：npm run dev
4. 首次访问将跳转 /auth，注册后登录进入系统。
